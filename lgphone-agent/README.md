# LGPhone Agent Tool

## Setup

1. Cai dat Node.js 18+ va Android Platform Tools (adb)
2. Chay `npm install`
3. Copy file `.env.example` thanh `.env`, dien `SUPABASE_SERVICE_ROLE_KEY`
4. Bat USB Debugging tren dien thoai, ket noi voi PC
5. Chay `npm start`

## .env

```
SUPABASE_URL=https://mcehliagbaheuqondqhd.supabase.co
SUPABASE_SERVICE_ROLE_KEY=your_service_role_key_here
```
